# ⚡ Hyper Dash

A simple endless runner game built with C++ and SDL2.

## 🎮 About
Hyper Dash is a fast-paced endless runner where the player must jump over obstacles and survive as long as possible.

## 🧱 Features (current)
- Player jump mechanic
- Obstacle system
- Basic collision detection

## 🚀 Tech Stack
- C++
- SDL2

## 🛠️ Build
Compile using g++ with SDL2 linked.
cmake --version


## 📌 Status
Early prototype (MVP)
